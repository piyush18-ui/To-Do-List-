# To-Do List Management System

This is a simple To-Do List project developed using C++ and Object Oriented Programming concepts.

## Features
- Add new task
- View all tasks
- Delete completed task
- File handling used for data storage

## Technologies Used
- C++
- OOP Concepts
- File Handling

## How to Run
1. Compile the code using any C++ compiler
2. Run the executable file
3. Use menu to manage tasks
